# Druid 部署

把 仓库-Docker (opens new window) 里的 docker-compose.yml (opens new window) 和 environment (opens new window) 拉取下来之后，

如果想改 UI 的端口，找到8888修改为58888即可

直接执行：

docker-compose -f docker-compose.yml up -d
1

就可以在端口 8888 看到 druid 的后台UI
