# 日记

我路途中见到了一张"斗转赛尔号"的卡片。这可以证明两件事：

- 吸引力法则强大

  周围的事物，只有你懂了，才能看得到周围的那些事物。

- 卡片是巨大的潜力市场

  如果能够和线上结合，类似 [Pokémon](https://www.pokemon.com/us/) 一样，那就是构建了一个虚拟的世界。结合着[卡牌驱动下的未来](../opinions/卡牌驱动下的未来.md) 的技能树，不断让孩子习得卡片上的相应技能。这对教育绝对是一个改变。
