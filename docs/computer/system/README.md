# 操作系统

## SimpleKernel
安装依赖

- 安装 `xcode`, [tool 'xcodebuild' requires Xcode](https://github.com/nodejs/node-gyp/issues/569)
- 安装 `xcode-select`
```
sudo xcode-select --install
sudo xcodebuild -license

```
- 安装 `cmake`,之后配置环境变量
